# Resume-HTML-Only
📌 Project Overview

This project is a simple resume created using pure HTML only.
It does not use any "<div>" tags or CSS, focusing completely on basic HTML structure and semantic elements.

🎯 Objective

The goal of this project is to understand and practice:

- Basic HTML tags
- Page structure without CSS
- Creating a readable layout using only HTML

🛠️ Technologies Used

- HTML5

✨ Features

- Clean and simple resume layout
- No use of "<div>" tags
- No CSS styling
- Beginner-friendly project
- Uses basic HTML elements like headings, paragraphs, and lists

📂 Project Structure

Resume-HTML-Only/
│── index.html
│── README.md
│── profile.jpg
